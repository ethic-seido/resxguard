# Changelog

## 0.1.0

Initial experimental release.

- Check and repair mojibake in `.resx` values, comments, and resource keys.
- Preserve common `.resx` encodings.
- Add safe `set` command for single key/value updates.
- Add CSV/JSON `import` command for batch updates.
- Add `--skip-missing` and `--continue-on-error` for imports.
- Add CSV/JSON reports.
- Add CLI color/emoji output and CI-friendly `--no-color --no-emoji` flags.
